# Javascript Parallax

## Usage

      new Parallax({
        container: '#animate',
        layers: [
          {selector: '.layer1', ratio: .003},
          {selector: '.layer2', ratio: .015},
          {selector: '.layer3', ratio: .030},
          {selector: '.layer4', ratio: .030},
          {selector: '.layer5', ratio: .03},
          {selector: '.layer6', ratio: .08},
        ]
      });
